Las herramientas que tienes que utilizar para un desarrollador backend:
 Conocer bien tu SO que utilizas día a día.
 Git y GitHub son dos cosas diferentes.
 Aprender bien la consola/terminal de tu SO
 Escoger el editor de texto y aprender a utilizarlo bien

Lo que aprendi es como utlizar los comandos de git, como contribuir en algun proyecto open sourse, saber cual es la diferencia de fork y clon.
Tambien vimos que nodejs es un modularizaión que corre con js del lado del servidor y que javaScript ocupa un avitualización V8.
En cuanto a paradigmas de progrmación conoci las clases, el constructor, funciones y las instancia, y como exportalo y como itilizarlo en otro proyecto utilizandor el require().
